1 Profile settings
The customer's account hub — completes the three sub-pages that were still being built in the previous release.
Capabilities:
Promo — A Promo screen inside Profile lists every promotion the customer can use right now. Tapping a promo applies its code to the cart and opens Products so the customer can shop with the discount already on.
Gift card — A Gift card screen shows every gift card the customer has redeemed with its face value, remaining balance, and validity. A "Redeem a gift card" button opens the redeem modal — enter the code, confirm, and the card lands in the list. Balance can then be spent at checkout.
Invite friends — A Referral screen explains the program in three quick steps, shows the customer's unique referral code with a Copy button and a Share sheet (WhatsApp / SMS / Email / other apps), and lists everyone the customer has already referred with reward status.

2 Notifications
The bell feed — one screen for every message the studio has sent the customer.
Capabilities:
All / Bookings / Payments tabs — Three tabs with a count pill on each so the customer sees at a glance how many unread messages sit in each bucket.
Today / Past groups — Notifications split into Today and Past, so the newest reminders always sit at the top.
Unread dots + tap to read — Unread messages carry a small dot on the left. Tapping any notification marks it read and deep-links to the booking, payment, or class it's about.
Mark all as read — A header action clears every unread dot on the current tab in one tap.

1 Profile
The instructor's own account page — personal details, work info, integrations, notification preferences, and security, all scoped to the logged-in instructor.
Capabilities:
Personal Information — Name, email, phone, introduction, work experience, and assigned class categories, with the instructor's shift hours and working days shown at a glance.
Branch Information — The branch the instructor is assigned to, with its address, working days, and opening hours.
Edit Profile — Update personal details and introduction from one screen; changes reflect on the instructor's schedule and dashboard right away.
Notifications — Turn each notification type on or off to control what the instructor receives.
Integrations — Connect the instructor's own tools (e.g. calendar) from the account page.
Security — Change password and review active sessions in a dedicated tab, kept separate from personal details so the page stays easy to scan.

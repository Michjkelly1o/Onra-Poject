Everything bookable now carries a type (Classes, Private sessions, Recovery & wellness), and "recovery" is its own type instead of a fake location. In the prototype you may still see some 404 pages; that means the page/module/feature is still being developed.

Interactive experience — This is a fully interactive prototype. Feel free to click, create, edit, and delete items to test the workflows.
Live updates — The prototype uses realistic sample data. Any changes you make apply live as you navigate between different screens.
Session persistence — Any data you create, edit, or cancel survives a page refresh and tab close. Your demo session sticks until you explicitly reset.

The big idea — a "type" dimension
Every bookable/scheduled thing is now exactly one of three types:
Classes — group sessions (built from a class template).
Private sessions — 1:1 training (a service).
Recovery & wellness — spa/wellness like massage, sauna, breathwork (also a service).
Recovery is no longer modelled as a separate "location". Location now means one thing only — a physical branch of the studio. The recovery area is a space (room) inside a real branch, not a location of its own. Type (what kind of session) and location (which branch) are cleanly separated everywhere.

1 Business & locations
Studio locations — each a real physical branch.
Capabilities:
No More "Spa" Location — The fake "Forma Spa" location is gone. The location list now shows only real studio branches (South, East, West). Recovery services relocated into Forma South.
No Club/Spa Scope — The "Location scope" (Club vs Spa) picker is removed from the branch form, and the "Location scope" row is gone from the branch detail page. A branch is just a branch.
Every Branch Can Have Rooms — "Add room" now appears on every branch. A new "Recovery Room" lives under Forma South for recovery/wellness sessions.

2 Services — Private sessions & Recovery & wellness
Creating a bookable service (1:1 private or recovery/wellness).
Capabilities:
Session Type Selector — When you create or edit a service, you now pick a type: Private session or Recovery & wellness (replaces the old "recovery" on/off toggle). Recovery reveals the "open sessions" option (multi-participant, no instructor); Private is always 1:1.
Optional Room Selector — Step 3 (Location) now lists every active branch plus an optional Room dropdown (rooms of the chosen branch, or "No room"). A session can use a room or not — your call.
Type Column — The services list swapped its "Recovery Yes/No" column for a clear "Type" column (Private session / Recovery & wellness).

3 Schedule
The daily operational calendar — one feed for classes, private sessions, and recovery.
Capabilities:
Coloured Type Tags — Every session on the schedule now shows a coloured type tag next to its name: green Class, purple Private, orange Recovery. It appears in the list view (its own sortable "Type" column), the day and week cards, and the click popover.
Three-Type Filter — The Type filter went from "Group / Appointments" to Classes / Private sessions / Recovery & wellness, filtering the whole calendar by session type.

4 Dashboard (Today tab)
Today's overview — now type-aware.
Capabilities:
Type Filter — A new session-type dropdown sits beside the location picker (All types · Classes · Private sessions · Recovery & wellness). Selecting one re-scopes the session tiles + the sessions list.
Today's Sessions (was "Today's classes") — The list now merges classes + private + recovery, each with its coloured type tag — so you see the whole floor, not just classes.
Occupancy Split — Occupancy is the one metric that can't collapse into a single number (class fill, private-slot utilisation and recovery capacity have different denominators). On "All" the occupancy tile shows a three-way split (mini bars: Classes / Private / Recovery); pick a type and it shows that type's single %.

General updates — Navigation restructure
Schedule Is Top-Level — The old "Classes" nav group is gone; Schedule is now its own daily item near the top of the sidebar.
Products & pricing — "Services & pricing" is renamed "Products & pricing" and holds Memberships & packages, Private sessions, and Recovery & wellness (the last two are type-filtered views of the services list, each with its own module title).
Classes Setup Section — A new "Classes" section (below the studio divider) groups the class-structure setup: Templates + Categories.

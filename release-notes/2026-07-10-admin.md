Note:

Interactive experience — This is a fully interactive prototype. Feel free to click, create, edit, and delete items to test the workflows.
Live updates — The prototype uses realistic sample data. Any changes you make apply live as you navigate between different screens.
Session persistence — Any data you create, edit, or cancel survives a page refresh and tab close. Your demo session sticks until you explicitly reset.

1 Dashboard
The owner's daily overview — today's activity, performance, and the customers and classes that need attention.
Capabilities:
Clearer Win-back Message — The win-back pop-up now reads simply: "No bookings since [date]. Holds [N] unused credits." — the reason a customer is flagged at a glance.
Sales By Product Shows Money — Hovering the Sales by product chart now shows the sales amount in AED, not the item count.
Ratings Only For Finished Classes — Upcoming and ongoing classes no longer show a rating (a class can't be rated before it happens); the Under-filled classes list drops the rating column entirely.
One Heading On Coming Up — Removed the duplicate "What's ahead" text under the Coming up tab so there's a single, clean heading.

2 Insights
Studio performance at a glance — KPI cards and trend widgets, now filterable by location.
Capabilities:
Cleaner Header — Removed the "Total KPIs" count line at the top so the page opens straight into the numbers.
Location Filter — Pick a specific branch to see that location's performance, or view all locations together.
Location-aware Widgets — Every chart and KPI card responds to the chosen location, so the whole page reflects one branch at a time.

3 Reports
The full reporting catalogue, grouped into categories, now with favourites.
Capabilities:
Favourite Reports — A star on the left of each report marks it as a favourite.
Pinned To The Top — Favourited reports jump to the top of their category, so the ones you use most are always first.
Saved Between Visits — Your favourites stay put after a refresh or closing the tab.
All Reports Complete — Every report in the catalogue is now built and available across all categories.

General updates
Breadcrumbs On Every Page — Each inner page now shows a small path under its title (e.g. "Customers > Customer details (name)") so you always know where you are and can jump back a level in one click.
Labels On Filter Dropdowns — Every filter dropdown now shows a short label above its options (Status, Date range, and so on) so it's clear what you're filtering.
Non-working Days In Grey — On Business & locations and staff schedules, non-working days now show in light grey instead of red, so they no longer look like an error.
Payroll Confirmation Renamed — The confirmation step when running payroll now reads "Payroll Details".
Tidier Delivery Hours — Removed the extra note box in the Delivery hours pop-up under Customer notifications for a cleaner screen.

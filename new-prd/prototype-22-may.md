1. Dashboard
Layout customization
Open Dashboard, walk through KPI cards + chart widgets pre-loaded for the Owner role
Click "Add widget" → side panel with searchable widget catalog
Search a widget by name → add it → widget lands in the grid
Click the ⋮ on any widget → "Remove widget" → confirmation modal → toast
Recent activity feed
Scroll to the activity feed → recent bookings & mock activity

2. Class template
Browse + filter
Open class templates → grid of template cards
Search by template name → results filter live
Open the side filter → filtering → Apply / Clear
Create
Click "Add template" → 2-step creation
Step 1: Basic info (name, description, banner image upload, category, class type, duration, capacity) + live preview card on the right
Step 2: Applicable memberships & packages — multi-select expandable card
Click "Create template" → toast notifications → returns to the list with new card
Detail page
Click any template → detail page opens with:
Banner + status badge + sidebar action menu
Tabs: Classes · Applicable Memberships · Applicable Packages
Show the Classes tab — sortable columns, search, and filter
Row ⋮ on a session → View details / Edit class / Cancel class
Edit + state changes (all via sidebar action menu in class template details)
Edit → opens the same 2-step form prefilled → save changes → toast
Archive → confirmation modal → status flips to Archived → toast
Recover an archived template → back to Active
On an Active template with sessions: Deactivate → can't delete because sessions exist → status flips to Inactive
Reactivate an inactive template → back to Active
On a never-used template: Delete option replaces Deactivate → permanent removal

3. Class schedule
Browse — 4 views
Switch between List, Day, Week, Month tabs
Day/Week/Month grids show color-coded class blocks by category
Class schedule state
Completed class: We can see all customer mark status & reviews, and can also delete the reviews.
Ongoing class: We can mark the customer as “present.”
Upcoming class over 24hrs: We can see the customers' list and still edit the class
Upcoming class under 24hrs: We can see the customers' list
Cancelled class: We can see the customer list in cancelled status, and we can’t do actions in this class.
Floating popup
Click any class in Day/Week/Month → floating popup with class info + status badge (Upcoming, Ongoing, Completed, Cancelled) + action icons (Add customer, Edit, Duplicate, Cancel) + "See details."
Search + filter (List view)
Search class schedule
Filter side panel
Add a class — 3-step flow
Click "Add Class"
Step 1: Class details (class template, gender, equipment, cover, capacity, duration) + live preview card
Step 2: Location & instructor — room picker shows room capacity
Spot selection toggle → opens overlay to customize spots (rows/columns + block individual spots).
Step 3: Date & time — Repeat (Does not repeat / Repeat weekly), Date picker, Start time. Recurring weekly with end condition & per-day time slots.
Duplicate a class
From popup → Duplicate → opens Add Class form pre-filled
Edit a class
From row ⋮ or popup → Edit class
Save → toast "Class updated"
Right preview panel reflects the actual class info
Add customer
From popup or class schedule details → Add customer → opens detail page with Add Customer drawer auto-expanded → search/select customer → assign credit from their active plan → confirm → customer lands in booked or waitlist tab
Buy a plan membership/credit package in POS for a customer that doesnt have any plan
Confirm payment when buying the plan for the customer
Cancel a class
Cancel class → destructive modal showing booked customer count → confirm → status flips to Cancelled, credits refunded → toast
Class detail page
Tabs: Booked · Waitlist · Cancelled · Reviews & Rating
Booked tab — roster with avatar, plan badge, spot, attendance status. Bulk select → "Mark present" or "Remove"
Mark a customer present → toast → attendance bar on the class template detail updates
Reviews tab — star ratings with comments, delete review action
